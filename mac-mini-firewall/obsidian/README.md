# MiniFW Obsidian Vault

Open this folder as an Obsidian vault:

**Folder:** `mac-mini-firewall/obsidian/`

## Start here

→ **[[Home]]** — Master build guide (walk through every phase)

## Folder structure

| Folder | Contents |
|--------|----------|
| `Build/` | Phases 0–6 (step-by-step) |
| `Network/` | Topology, Google Fiber, Deco XE75 |
| `Operations/` | Rollback, triage, emergency card |
| `Reference/` | USB, preflight, IPs, commands, configs |

## Tips

- Use **Graph view** to see how notes connect
- **Home** is the map of content (MOC) — every build step links from there
- Plain markdown copies also live in `docs/` and on the **MINIFWSETUP** USB

## Also on the USB stick

The `build-setup-usb.sh` script copies this vault to the MINIFWSETUP stick under `obsidian/`.
