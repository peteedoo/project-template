# Phase 0 — Before You Start

← [[Home]] · Next → [[Phase 1 - Install Ubuntu]]

⏱ ~30 minutes · **Do not skip this phase**

---

## Shopping / hardware checklist

| Item | Required | Notes |
|------|----------|-------|
| USB 3.0 Gigabit Ethernet adapter | ✅ | Mac Mini has only one built-in port |
| Gigabit switch (5-port) | Recommended | Deco + Mac Mini LAN + wired devices |
| USB keyboard + HDMI monitor | ✅ | Borrow if needed |
| Spare Ethernet cables | Recommended | Label before unplugging |
| Phone with cellular data | ✅ | Backup internet + GF support |
| 8 GB+ USB stick (MINIFWSETUP) | ✅ | See [[MINIFWSETUP USB]] |
| Separate USB for Ubuntu install | ✅ | See [[Phase 1 - Install Ubuntu]] |

---

## Photograph current wiring

Take photos of:

1. [[Google Fiber and Deco XE75#Your equipment (from photos)|Fiber Jack → Network Box]] cable (which port on each end)
2. Network Box back panel (all ports)
3. Network Box LED status (all green = healthy baseline)
4. Deco placement and what's plugged in

Store on your phone. Needed for [[Backup and Rollback]].
